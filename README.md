# EMS - Education Management System

Hệ thống quản lý giáo dục thông minh và hiện đại, giúp tối ưu hóa quy trình dạy và học cho Giáo viên, Học sinh và Quản trị viên.

## 🎨 Design System
Dự án sử dụng bộ nhận diện thương hiệu cao cấp với tông màu:
- **Primary**: `#355872` (Deep Blue)
- **Secondary**: `#7AAACE` (Medium Blue)
- **Accent**: `#9CD5FF` (Azure/Sky Blue)
- **Background**: `#F7F8F0` (Cream/Off-white)

## 🏗️ Kiến trúc Layout
Hệ thống sử dụng kiến trúc đa Layout (Multi-layout) linh hoạt:
- **MainLayout**: Dành cho Landing Page, About, Contact (có Header/Footer).
- **BlankLayout**: Dành cho các trang Authentication (Đăng nhập, Đăng ký) và Error.
- **DashboardLayout**: Dành cho cổng quản trị chuyên nghiệp (có SideMenu).

## 🚀 Công nghệ sử dụng
- **Core**: ReactJS + Vite
- **Styling**: Tailwind CSS v4 + Vanilla CSS
- **Routing**: React Router DOM v6
- **UI Components**: Custom reusable components (Button, Inputs, Layouts)

## 🛠️ Cài đặt và Chạy dự án
1. Cài đặt dependency: `npm install`
2. Chạy môi trường development: `npm run dev`
3. Build production: `npm run build`
